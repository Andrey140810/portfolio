# Портфолио Frontend-разработчика

Современный сайт-портфолио с анимациями и адаптивным дизайном.

## 🚀 Технологии

- **React 19.1.1** - современная версия React
- **Vite** - быстрый сборщик
- **Framer Motion** - библиотека анимаций
- **React Icons** - иконки

## 📦 Установка

```bash
npm install
```

## 🏃 Запуск проекта

```bash
npm run dev
```

Проект будет доступен по адресу `http://localhost:5173`

## 🏗️ Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist`

## 📁 Структура проекта

```
portfolio/
├── src/
│   ├── components/      # React компоненты
│   │   ├── Navbar.jsx
│   │   ├── Hero.jsx
│   │   ├── About.jsx
│   │   ├── Skills.jsx
│   │   ├── Projects.jsx
│   │   └── Contact.jsx
│   ├── styles/          # Стили компонентов

│   ├── assets/          # Изображения и ресурсы
│   ├── App.jsx          # Главный компонент

```

## ✨ Особенности

- 🎨 Современный дизайн с темной темой
- 🎭 Плавные анимации и переходы
- 📱 Полностью адаптивный дизайн
- ⚡ Быстрая загрузка благодаря Vite
- 🔗 Интеграция с GitHub API
- 🎯 Оптимизация для SEO

## 📝 Секции сайта

1. **Hero** - Главная секция с приветствием
2. **About** - О себе и опыте
3. **Skills** - Технологический стек
4. **Projects** - Проекты с GitHub
5. **Contact** - Форма обратной связи

## 🎨 Кастомизация

Все цвета и стили определены в CSS переменных в `src/index.css`:

```css
--primary-color: #6366f1;
--secondary-color: #8b5cf6;
--accent-color: #ec4899;
```
