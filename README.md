# Портфолио Frontend-разработчика

Современный сайт-портфолио с анимациями и адаптивным дизайном.

🌐 **Live Demo**: [https://andrey140810.github.io/portfolio/](https://andrey140810.github.io/portfolio/)

## 🚀 Технологии

- **React 19.1.1** - современная версия React
- **Vite** - быстрый сборщик
- **Motion (React)** - библиотека анимаций
- **React Icons** - иконки

## 📦 Установка

```bash
npm install
```

## 🏃 Запуск проекта

```bash
npm run dev
```

Проект будет доступен по адресу `http://localhost:5173`

## 🏗️ Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist`

## 📁 Структура проекта

```
portfolio/
├── src/
│   ├── components/      # React компоненты
│   │   ├── Navbar.jsx
│   │   ├── Hero.jsx
│   │   ├── About.jsx
│   │   ├── Skills.jsx
│   │   ├── Projects.jsx
│   │   └── Contact.jsx
│   ├── styles/          # Стили компонентов
│   │   └── components.css
│   ├── assets/          # Изображения и ресурсы
│   ├── App.jsx          # Главный компонент
│   ├── App.css          # Стили приложения
│   ├── index.css        # Глобальные стили
│   └── main.jsx         # Точка входа
├── public/              # Публичные файлы
├── .github/workflows/   # GitHub Actions для деплоя
└── package.json
```

## ✨ Особенности

- 🎨 Современный дизайн с серой темой
- 🎭 Плавные анимации и переходы
- 📱 Полностью адаптивный дизайн

## 📝 Секции сайта

1. **Hero** - Главная секция с приветствием
2. **About** - О себе и опыте
3. **Skills** - Технологический стек
4. **Projects** - Проекты с GitHub
5. **Contact** - Форма обратной связи

## 🚀 Деплой на GitHub Pages

Сайт автоматически деплоится на GitHub Pages при пуше в ветку `main` через GitHub Actions.
